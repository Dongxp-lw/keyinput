## 5. 缺陷处理

- 任何明文秘密进入日志都是发布阻断问题。
- 任何导入或同步过程中的保险库损坏都是发布阻断问题。
- 任何要求登录才能本地使用的回归都是发布阻断问题。
- 任何离线流程中的隐藏网络依赖都是发布阻断问题。
- 平台特定键盘限制只有在形成文档并提供备选路径时才可接受。
