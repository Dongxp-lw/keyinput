# 路线图

## 阶段 0：产品与平台验证

- 验证 Android 输入法插入行为。
- 验证 iOS 键盘扩展限制和共享容器行为。
- 验证 HarmonyOS 输入法 API 和签名要求。
- 选择第一个实现平台。
- 确认初始加密库和保险库文件格式。

## 阶段 1：离线 MVP

- 实现本地保险库和核心条目模型。
- 实现安全键盘选择流程。
- 实现主密码和生物识别解锁。
- 增加加密导出/导入。
- 增加基础 TOTP 支持。
- 增加发布测试清单。

## 阶段 2：迁移工具

- 实现加密迁移包。
- 增加本地设备配对或基于二维码的迁移。
- 增加迁移校验和恢复行为。
- 探索桌面助手或 CLI 工具。

## 阶段 3：可选云服务

- 增加用户账号。
- 增加第一方加密云同步。
- 增加设备管理。
- 增加版本历史。
- 增加订阅权益。

## 阶段 4：高级输入

- 增加字段模板。
- 如果产品策略批准，增加可选的非私密词库模式。
- 增强保险库组织和共享能力。
- 增加家庭/团队流程。
