# 贡献者行为准则

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、可见或不可见的身心障碍、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向如何，参与本项目和社区的每个人都能享有免于骚扰的体验。

我们承诺以有助于建立开放、友善、多元、包容和健康社区的方式行事与互动。

## 我们的准则

有助于为社区营造积极环境的行为包括：

- 对他人展现同理心与善意；
- 尊重不同的意见、观点和经验；
- 给予并优雅地接受建设性反馈；
- 为自己的错误承担责任、向受影响者致歉，并从中学习；
- 关注的不仅是个人，而是整个社区的最大利益。

不可接受的行为包括：

- 使用带有性意味的言语或图像，以及任何形式的性关注或性骚扰；
- 挑衅、侮辱或贬损性的评论，以及人身或政治攻击；
- 公开或私下的骚扰；
- 未经明确许可，公布他人的私人信息（如住址或电子邮箱）；
- 在专业场合中可合理认定为不当的其它行为。

## 执行责任

社区维护者有责任阐明并落实可接受行为的标准，并对其认为不当、具威胁性、冒犯性或有害的任何行为，采取适当且公正的纠正措施。

社区维护者有权也有责任删除、编辑或拒绝不符合本准则的评论、提交、代码、wiki 编辑、issue 及其它贡献，并在适当时说明理由。

## 适用范围

本准则适用于所有社区场合，也适用于个人在公开场合代表本社区之时，例如使用项目的官方邮箱、通过官方社交账号发布，或在线上线下活动中作为指定代表。

## 执行

如遇滥用、骚扰或其它不可接受的行为，请通过 [项目配置](project-config.toml) 的 `[contact].conduct_email`（发布前填写）向社区维护者举报。所有投诉都会被及时、公正地审查与调查。

社区维护者有义务为举报人保密。

## 执行准则

社区维护者将依据以下「社区影响准则」，判定对违反本准则行为的处理：

1. **纠正**——*社区影响*：使用不当言语或其它被认为不专业、不受欢迎的行为。*处理*：私下书面警告，说明违规性质并解释行为为何不当；可要求公开致歉。
2. **警告**——*社区影响*：单次或一系列行为构成的违规。*处理*：警告并附后果。在指定时间内，不得与相关人员互动，包括主动与准则执行者互动；违反可能导致封禁。
3. **临时封禁**——*社区影响*：严重违规，包括持续的不当行为。*处理*：在指定时间内禁止与社区进行任何形式的互动或公开交流；违反可能导致永久封禁。
4. **永久封禁**——*社区影响*：表现出违规模式，包括持续不当行为、骚扰个人，或敌视、贬低某类群体。*处理*：永久禁止在社区内进行任何形式的公开互动。
